* 재부팅 후에도 방화벽 정책 유지 
  ① 패키지 설치: 
  $ sudo apt-get install iptables-persistent  (or netfilter-persistent)
  ② 방화벽 정책을 저장: 
  $ sudo netfilter-persistent save
  ③ 방화벽 정책을  다시 로드: 
  $ sudo netfilter-persistent start
